CloudCue is a full-stack cloud workspace app built with Next.js and Node.js, featuring Kanban-style project management, a Notion-like page editor, real-time collaboration, and direct file uploads to S3-compatible storage (MinIO). It is Dockerized with PostgreSQL, Prisma, Socket.io, and Nginx, making it a self-hostable productivity platform with JWT auth, comments, mentions, and dark/light mode support.
